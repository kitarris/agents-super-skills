# Visual System Rules

## Principles

- Typography is role-based, not viewport-width-based.
- Title/key insight may be larger; content slides must be calmer.
- Body text must survive screen sharing.
- Accent color guides attention and stays under 10% of slide area.
- Background texture must never fight text.
- Voxel/3D is optional, not a default requirement.

## Typography Defaults

```text
title slide headline: 48-68px desktop
content slide headline: 34-44px desktop
body/support: 18-24px desktop
card title: 18-24px
card body: 15-18px
headline -> paragraph gap: 28-44px
paragraph -> visual/note gap: 24-44px
```

## Color Rules

- Text contrast target: high enough for screen sharing.
- Use one primary accent and one caution accent.
- Avoid all-cyan, all-purple, all-slate, and all-beige themes.
- Use gradients only when they encode emphasis or data, not decoration.

## Visual Proof Rule

```text
If visual style conflicts with proof clarity, proof clarity wins.
```
